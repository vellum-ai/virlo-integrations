# Results Viewer

The Results Viewer is a visual app for browsing the findings of a Content
Research Agent — videos, rising creators, hashtags, and sounds — instead of
reading raw data. Use this reference to explain the app when a user asks what it
is or how to use it.

## What the user sees

An interactive panel with four tabs:

- **Top Videos** — the best-performing videos in the niche, as cards with
  thumbnail, platform, creator, view/like/share counts, and format/topic tags.
  Each card links out to the original video.
- **Creator Outliers** — rising creators who are punching above their follower
  count, with their key stats and content angle. Each card links to the
  creator's profile.
- **Hashtags** — the niche's hashtags ranked by reach, with a bar for each.
- **Rising Sounds** — trending audio tracks to watch.

## How the user opens it

1. Ask for research on a niche (e.g. *"what's going viral in London fitness?"*).
   This kicks off a Content Research Agent, which takes ~15-20 minutes.
2. When the run finishes, the assistant opens the Results Viewer for that agent —
   the user doesn't need to do anything.
3. To reopen results later, the user can open the app and paste the agent's ID,
   or the assistant can reopen it for them.

## Answering common questions

- **"How do I see my results?"** — The assistant opens the Results Viewer
  automatically once a run finalizes; results can be reopened any time.
- **"Does viewing results cost anything?"** — No. Browsing and refreshing
  results is always free; only starting new research costs credits.
- **"Can I open an old search?"** — Yes. Every agent keeps its results, so the
  viewer can be reopened for any past agent by its ID.
- **"Where does the data come from?"** — It's the live output of the user's own
  Content Research Agent, pulled straight from Virlo.
